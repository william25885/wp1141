# Final Project – Midterm Plan

## 1. 本次 Prototype 已完成
- 基本頁面架構
- 登入框架
- 兩個示意頁面

## 2. 最終版本預計完成項目
- 完整的使用者流程
- 資料庫串接
- 三大主要功能：
  - **聚會管理系統**：發起、參與、取消聚會，支援多種聚會類型（午餐、咖啡、晚餐、喝酒、語言交換），語言篩選功能
  - **即時通訊系統**：一對一私人聊天、聚會群組聊天室、在線狀態顯示、訊息歷史記錄
  - **好友系統**：搜尋並添加好友、好友請求管理（發送、接受、拒絕）、好友列表與在線狀態、查看其他用戶的公開個人資料

- **Week 1 (11/13 - 11/20)**：完成基本架構與認證系統

  - 私人聊天功能（一對一即時通訊）
