## 網站特色

- **深色主題設計** - 用以搭配夜景的視覺效果
- **半透明元素** - 個人資料卡片採用毛玻璃效果
- **流暢動畫** - 元件浮現動畫和互動效果
- **響應式設計** - 完美適配桌面、平板和手機
- **TypeScript 開發** - 型別安全的現代 JavaScript
- **技能展示** - 動態進度條顯示程式技能
- **簡潔導航** - 深藍色導航欄與背景完美融合

##  設計理念

- 因為在選課截止前幾分鐘才加了這堂課，一進來就看到有個作業要截止了...，於是很匆忙的搞了一個簡單的純前端個人網站，內容大概是很簡單的介紹一下自己，跟放些一兩張和自己有關的照片。

- 技能專長的部分是大概 google 了一下看到很多人都會放些自己的經歷專長等，也就很簡單的做了一個類似的，最後再補上一些簡單的聯絡方式。

- 剩下實在是不太知道還能放些什麼跟自己太有關的東西，就加了一些簡單的美編！

## 專案結構

```
hw1/
├── index.html          # 主要 HTML 檔案
├── styles.css          # CSS 檔案
├── script.ts           # TypeScript
├── script.js           # 編譯後的 JavaScript 檔案
├── images/             # 圖片資源
│   ├── background-photo.jpg  # 首頁背景圖
│   └── personal_pic.jpg      # 個人照片
└── README.md          # 專案說明文件
```

## 快速開始

### 1. 由於 js 檔為 Typescript 檔編譯而來，編譯方式為：

```bash
tsc script.ts
```

### 2. 若無 Node.js，則須先下載，並執行：

```bash
sudo npm install -g typescript
```

以安裝 typescript



### 3. 若已看到 js 檔，點擊 html 即可開啟視窗
